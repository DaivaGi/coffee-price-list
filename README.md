<a href="https://coffee-1cc19.firebaseapp.com">Live Demo</a>

<h3>Build/Run</h3>
<h4>Requirements<h4>
<p>Node.js</p>
<p>NPM</p>
  <br>
<p>/* First, Install the needed packages */</p>
  
<code>npm install</code>
<br>
<p>/* Then start both Node and React */</p>
<code>npm start</code>
<br>
Open http://localhost:3000 to view it in the browser.
